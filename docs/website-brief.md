# 扬州精艺机电有限公司官网开发任务文档 v1.0

## 1. 项目目标

为扬州精艺机电有限公司开发一个专业的企业官网第一版，网站定位为：

**连接器行业数控专用机床与精密加工解决方案供应商**

网站需要突出公司在光电连接器、电连接器、航空航天电子连接器加工设备领域的专业能力，展示公司介绍、产品设备、应用方案、研发专利、资质荣誉和联系方式。

网站第一版目标不是做复杂后台，而是完成一个可上线、可展示、可询盘的静态/半静态企业官网。

## 3. 网站技术建议

请优先使用以下技术栈：

- Next.js 或 React + Vite
- TypeScript
- Tailwind CSS
- 响应式布局，兼容 PC、平板、手机
- 图片资源统一放在 `/public/assets`
- 产品数据统一抽离到 `/src/data/products.ts`
- 页面组件化，便于后续增加产品和多语言
- 第一版不需要复杂后台
- 联系表单第一版可以先做前端表单 UI，不需要真实发送邮件；但代码中预留 API 接口结构

## 4. 网站整体风格

视觉方向：工业感、专业感、可信赖、简洁高级。

推荐配色：

- 主色：深蓝 / 工业蓝
- 辅助色：深灰 / 银灰
- 强调色：科技蓝或橙色
- 背景色：白色、浅灰、深蓝渐变

页面气质：不要做成普通企业黄页。整体风格应接近高端制造企业官网，突出设备、参数、应用场景和技术能力。

图片要求：

- 工厂外景
- 机床设备图
- 产品样件图
- 资质证书图
- 专利证书图
- 连接器应用图

如果图片不够清晰，先用占位图结构，但文件命名和图片位置要预留好。
